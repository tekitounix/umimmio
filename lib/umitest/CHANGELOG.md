## [0.1.0] - 2026-02-07

### Added

- Initial release with `Suite` and `TestContext` classes.
- Zero-macro design using `std::source_location`.
- Header-only, no-exception, no-heap architecture for embedded compatibility.
- `assert_eq`, `assert_true`, `assert_false`, `assert_near` assertions.
- Standardized library structure (UMI_LIBRARY_STANDARD compliant).
- CI workflow for host tests.
